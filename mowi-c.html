MOWI-C är ett gränssnittsbibliotek (UI Library) för windows och har en text baserat stil, fast är inte gjort för kommandotolken.

Det är egentligen lite underligt varför det heter just "mowi-c", men det är för idéen har sitt ursprung i ett helt annat projekt. "mowi" står för "MOvable WIndows" alltså "flyttbara fönster" vilket kanske passar någorlunda för vad mowi är nu men ursprungligen var programmet mer som ett textbaserat program där det fanns flyttbara fönster med användargränssnitt. För att göra dessa "fönster" så skapas egentligen det som nu är mowi, ett system för att skapa gränssnitt.

Systemet för att skapa dessa "fönster" var det som sedan separerades till det som nu är mowi, alltså utan flyttbara fönster. Allt detta var hitils propgrammerat i Java men när jag började använda C så skrev jag om mowi, som en utmaning, därav "mowi-c".
